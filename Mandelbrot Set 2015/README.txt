1. Save Makefile and skeleton.c in the same directory.

2. Open the terminal and write: make

3. To run the program, write: ./main

4. Choose between 0 (set the size of the picture by yourself) and 1 (set the size of the picture by default). If you try to type anything else, you will get an error-message.

5. Choose between 0 (just generates the picture of the Mandelbot set) and 1 (same as 0 but it also zooms into a point).

6. Quit program by pressing ESC.
