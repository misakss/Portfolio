1. Save the file Project.m in the same place as the pictures of the coins are.

2. Type "run Project1" in commando window.

3. Choose which of the three pictures you would like to use. 
If you type anything other than 1, 2 or 3 you will get an error message and you can try again.

4. In the commando window you can follow the process of the program. 
After the program is done you will see the result and how long it took for the program to run.
